# CALCULADORA DE CONSUMO ELÉTRICO

## Objetivo do Projeto
Neste sitema, é possível calcular quanto que um aparelho gasta de energia elétrica por mês. 

## Linguagem Utilizada
![python](https://img.shields.io/badge/language-python-black?style=flat&logo=python&logoColor=3776AB&labelColor=white&color=3776AB)

## Calculos
- ConsumoMensal = (Potencia * Tempo * 30) / 1000
- ValorFixo = 0.75
- CustoEstimado  = ConsumoMensal * ValorFixo

## Instruções
Para usar este programa, apenas se deve colocar o nome do aparelho que quer calcular, a potência em watts (W) e o tempo médio de uso diário em horas. Após colocar essas informações, o resultado  dos calculos irá aparecer.
